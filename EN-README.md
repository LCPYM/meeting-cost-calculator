# Meeting Cost Calculator

## 📊 Project Overview

Meeting Cost Calculator is an open-source real-time meeting cost tracking tool that helps teams and organizations quantify the actual cost of meetings and cultivate time management awareness.

**Website:** https://meeting-cost-calculator.vercel.app  
**Developer:** LCPYM  
**License:** MIT License

---

## ✨ Core Features

### 1. Real-Time Cost Tracking
- Input number of attendees, average hourly rate, and estimated duration
- Live timer displaying accumulated costs
- Multi-currency support (HKD, USD, EUR, GBP, CNY, JPY)

### 2. Budget Alert System
- Set meeting budget limits
- Automatic alerts when over budget
- Progress bar showing real-time budget usage

### 3. Full-Screen Presentation Mode
- Perfect for conference room displays
- Clear real-time cost visualization
- Adjustable scale and spacing

### 4. Meeting Records & Analytics
- Automatic meeting history storage
- Statistics: total meetings, total cost, average duration
- CSV export for reports

### 5. Template System
- Save frequently used meeting configurations
- Quick template loading
- Improved workflow efficiency

### 6. Multilingual Support
- Traditional Chinese / English
- Language switching without data loss

### 7. Three Color Themes
- Ocean Blue
- Purple Dream
- Nature Green
- Dark Mode

### 8. Sharing Features
- Download meeting cost screenshots
- Copy settings link
- QR Code for quick sharing
- CSV report export

## 💻 Technical Highlights

- Pure frontend implementation (HTML + CSS + JavaScript)
- No backend server required
- All data stored locally in browser
- Responsive design for mobile and tablet
- Deployed on Vercel with global CDN

---

## 🚀 Quick Start

1. Visit https://meeting-cost-calculator.vercel.app
2. Enter number of meeting attendees
3. Set average hourly rate
4. Click "Start Meeting Timer"
5. Watch costs accumulate in real-time
