# Meeting Cost Calculator - 會議成本計算器

## 📊 項目簡介

Meeting Cost Calculator 是一個開源的即時會議成本計算工具，幫助團隊和組織量化會議的實際成本，培養時間管理意識。

**網站：** https://meeting-cost-calculator.vercel.app  
**開發者：** LCPYM  
**授權：** MIT License

---

## ✨ 核心功能

### 1. 即時成本計算
- 輸入參與人數、平均時薪、會議時長
- 計時器即時顯示累計成本
- 支援多種貨幣（HKD, USD, EUR, GBP, CNY, JPY）

### 2. 預算警示系統
- 設定會議預算上限
- 超支時自動警示
- 進度條即時顯示預算使用情況

### 3. 全螢幕演示模式
- 投影會議室大螢幕
- 清晰的即時成本顯示
- 可調整縮放比例和疏密度

### 4. 會議記錄與分析
- 自動儲存會議歷史
- 統計總會議次數、總成本、平均時長
- 匯出 CSV 報表

### 5. 範本系統
- 儲存常用會議設定
- 快速載入範本
- 提升工作效率

### 6. 多語言支援
- 繁體中文 / English
- 語言切換不影響數據

### 7. 三種配色主題
- Ocean Blue（海洋藍）
- Purple Dream（紫色夢幻）
- Nature Green（自然綠）
- Dark Mode（深色模式）

### 8. 分享功能
- 下載會議成本截圖
- 複製設定連結
- QR Code 快速分享
- 匯出 CSV 報表

## 🚀 快速開始

1. 訪問 https://meeting-cost-calculator.vercel.app
2. 輸入會議參與人數
3. 設定平均時薪
4. 點擊 "Start Meeting Timer"
5. 即時觀看成本累積
